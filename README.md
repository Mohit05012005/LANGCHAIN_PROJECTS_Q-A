### HOW TO INSTALL AND RUN:
     #  CLONE THE REPOSITORY USING : git clone https://github.com/Mohit05012005/LANGCHAIN_PROJECTS_Q-A.git
     cd LANGCHAIN_PROJECTS_Q-A
     # CREATE VIRTUAL ENVIRONMENT : python -m venv venv venv/Scripts/activate   # Windows
     # INSTALL DEPENDENCIES: pip install -r requirements.txt
     # ADD ENVIRONMENT VARIABLE : GROQ_API_KEY=your_key   AND  HF_API_KEY=your_token
     # RUN THE APP: streamlit run app.py
